# Process a Single PDF for Post-Render Title Enrichment

Process a Single PDF for Post-Render Title Enrichment

## Usage

``` r
process_single_pdf_post_render(pdf_path, gs_bin)
```

## Arguments

- pdf_path:

  Path to the PDF file.

- gs_bin:

  Ghostscript binary name, or `NULL` if not available.

## Value

Invisible `NULL`.
