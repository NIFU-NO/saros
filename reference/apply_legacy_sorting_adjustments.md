# Apply legacy sorting adjustments for special cases

Apply legacy sorting adjustments for special cases

## Usage

``` r
apply_legacy_sorting_adjustments(
  data,
  indep_names = character(0),
  translations = list()
)
```

## Arguments

- data:

  Dataset

- indep_names:

  Independent variable names

- translations:

  Translation strings

## Value

Dataset with legacy adjustments applied
