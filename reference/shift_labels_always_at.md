# Shift labels_always_at

Shift labels_always_at

## Usage

``` r
shift_labels_always_at(data, labels_always_at = NULL, after = Inf)
```

## Arguments

- data:

  Dataset

- labels_always_at:

  Labels to move to bottom or top

- after:

  Position to move labels to (0 = top, Inf = bottom)

## Value

Dataset with data\$.variable_label adjusted
